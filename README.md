
# ☕ Coffee Shop Sales Dashboard (Power BI)

## 📌 Project Overview
This project is a Power BI dashboard built to analyze and visualize sales performance of a coffee shop. It provides insights into sales trends, product performance, and customer behavior using interactive visualizations.

The goal of this project is to demonstrate data analysis, data modeling, and dashboard design skills.

---

## 📊 Key Features
- 📈 Sales trend analysis (Daily / Monthly / Yearly)
- 🧾 Total revenue, quantity sold, and order count KPIs
- 🥤 Top-selling products and categories
- 🏪 Store/location-wise performance
- ⏰ Time-based insights (peak hours, best days)
- 📍 Customer behavior insights

---

## 🛠 Tools & Technologies Used
- **Power BI**
- Data Cleaning & Transformation using **Power Query**
- Data Modeling (Star Schema)
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Description
The dataset contains:
- Order details
- Product information
- Store/location data
- Date & time data

---

## 🧠 Key Insights
- Identified top-performing products contributing to revenue
- Found peak sales hours and high-demand days
- Analyzed store-wise performance differences
- Improved understanding of customer purchase patterns
